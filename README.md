# hack-secretsanta
This is the repository for the internal Kin Opt-In tool, previously named 'Secret Santa'


## Getting started:
- Clone the repo
- `cd hack-secretsanta`
- `npm i`
- `npm run build`
- `npm run start`
- Browse to 0.0.0.0:8080/{campaign_name}/
